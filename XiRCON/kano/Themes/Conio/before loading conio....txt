³ How do I change my colors?                                 ³°°
³    conio.thm colors are set with six /conio variables.     ³°°
³    These variables are:                                    ³°°
³                                                            ³°°
³    bright          Bright color                            ³°°
³    dark            Dark color                              ³°°
³    white           Color for white                         ³°°
³    norm            Normal text                             ³°°
³    gray            Dark gray color                         ³°°
³    back            Background color                        ³°°
³                                                            ³°°
³    The values of these variables should be a mIRC/XiRCON-  ³°°
³    style color number (00-15).  To see a list of all       ³°°
³    colors and their corresponding numbers, use the /c      ³°°
³    command.                                                ³°°
³                                                            ³°°

³ What are presets?                                          ³°°
³    Presets are preset values for /conio variables which    ³°°
³    can be used to set some or all variables with one easy  ³°°
³    command.  For instance, a particular preset "purple"    ³°°
³    might set the conio.thm color variables to shades of    ³°°
³    purple; whereas the "default" preset would set all      ³°°
³    variables back to their default state.                  ³°°
³                                                            ³°°
³    To see a list of all presets, type /preset.  To use a   ³°°
³    particular preset, type /preset name.                   ³°°
³                                                            ³°°

³ Where are presets and styles located?                      ³°°
³    Presets and styles are located in .sty files which      ³°°
