1  Type /theme to load a theme. It will auto-load from then on until you decide
   to use another one.

2  To load an addon (*.ka) stick it in your_xirc_dir\addons (type /reload to
   activate it.)

3  A help file is in your XiRCON\kano directory

4  As a reward for reading the readme... To stop autojoining #kano type
   /ajoin.

5  There are MORE ADDONS in 'xircon\kano\MORE ADDONS.' these do things like
   bot control and er, other stuff. You may have chosen to install these
   automatically.

6  Get the latest betas, addons, and other cool stuff by typing /kfile.

7  For more information, try www.kano.net.
